# Design Tokens

Design tokens são os valores fundamentais que definem o sistema de design.
Eles garantem consistência visual em todos os componentes e aplicações.

## Como usar

Os tokens estão disponíveis de duas formas:

### 1. CSS Variables

```css
.my-component {
  background-color: var(--color-primary-500);
  padding: var(--spacing-4);
  border-radius: var(--border-radius-md);
}
```

### 2. JavaScript/TypeScript

```typescript
import { colors, spacing } from '@prototipo/tokens';

const styles = {
  backgroundColor: colors.primary[500],
  padding: spacing[4],
};
```

## Breakpoints

Sistema de breakpoints responsivos.

  {Object.entries(tokens.breakpoints).map(([name, value]) => (

        {value}
      </div>
      <div style={{ fontSize: '0.75rem', color: '#6b7280' }}>
        {name === 'sm' && '≥ Small devices (landscape phones)'}
        {name === 'md' && '≥ Medium devices (tablets)'}
        {name === 'lg' && '≥ Large devices (desktops)'}
        {name === 'xl' && '≥ Extra large devices (large desktops)'}
        {name === '2xl' && '≥ 2X large devices (larger desktops)'}
      </div>
    </div>
  ))}
</div>
